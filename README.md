## Getting Started

To get a local copy of this app you can use :
```
git clone  https://github.com/moise-mulungu/Air-pollution
```
# Install

- Run :
1. `npm create-react-redux.`
2. `cd <app's name>`,
3. `npm start`

# Usage

Run `npm start` to open the live server
